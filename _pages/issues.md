# 問題回報

如果您在使用台語齒盤時遇到任何問題，歡迎透過以下方式回報：

## GitHub Issues

請到我們的 GitHub 專案頁面回報問題：

[前往 GitHub Issues](https://github.com/taigikeyboard/taigikeyboard/issues)

## 回報前請確認

在回報問題前，請先確認：

1. **搜尋現有問題**：您遇到的問題可能已經有人回報過
2. **提供詳細資訊**：
   - iOS 版本
   - 裝置型號
   - 問題的重現步驟
   - 截圖（如果可能的話）

## 功能建議

我們也很歡迎您提供功能建議！請在 GitHub Issues 中標記為 "enhancement"。
